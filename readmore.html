So the way we make this one work is - first, we set these hidden paragraphs with the class name "more" which is set in our CSS to display: none.

We also gave our paragraph above that says "read more" the class "toggle" (we could use an ID here since it's unique on this page AND we want to make sure that when it's clicked, no other "read more" buttons get triggered, so it would be a good idea to use a unique ID here instead of a class but I didn't).

In our JavaScript, we collected all our elements with the class "toggle" into the variable togg (just one) and then we collected all our elements with the class "more" into an array called "mores" (there are several).

I've also given the "read more" paragraph an "onClick" event that calls the JavaScript "readMore" function.

Because I want this button to be a "read more" button that then turns into a "read less" button, the first thing our function does is check to see what the .innerText of the paragraph says. if it says "read more" then I use a for loop to iterate through the mores array and change the display from none to block and I change the innerText of the "toggle" paragraph to "read less."

Otherwise (else), I use a for loop to iterate through the mores array and change all their displays to "none" and I change the innerText of the "toggle" paragraph to "read more."

I'm sure there are other ways to do this, but this is the way I did it.
