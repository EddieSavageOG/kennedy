Creating an Overlay on Top of an Image

The overlay above uses the ::before selector to style a white overlay on top of the image.

To get this effect, I need to put my image inside a div (well, I don't NEED to, but it makes life easier if I do) and I've used assigned the class name "overlay" to that div.

I styled my image simply by assigning it a fixed pixel width (because overlays are finicky). You can use rems or percentages, but it was making me to do too much math so I went with pixels. Normally, I'd stick to rems or percentages because that makes my sites naturally responsive but I'm being lazy today, so it's pixels. :D

I've assigned text-align: center to my div to center my image AND my overlay.

The overlay itself is the MOST styled element on this page (outside the navigation of course). The styling looks like this:


      .overlay::before {
            background-color: #fff;
            opacity: .5;
            content: "";
            z-index: 1;
            position:absolute;
            width: 750px;
            height: 500px;
            display: inline-block;
      }

The content property is necessary to make before renderable, otherwise it has no content and won't exist. The z-index is necessary to make sure the overlay is on top of the image. The position is set to absolute so that the overlay can be positioned on top of the image. The width and height are set to the same width and height as the image so that the overlay covers the image. The display is set to inline-block so that the overlay is rendered as a block element. And position is set to absolute to take it out of the normal flow of standard rendering. Otherwise it wants to render as a block element and pushes the image down below it.

And there you have it! How to use ::before to create an overlay. You can apply this to text overlays, too, if you want, by adding actual words to the content property and removing the background color (or keeping it - your choice!).
